# My Portfolio Website

A responsive personal portfolio website showcasing my profile, skills, projects, education, and contact information.

## Features

- Responsive design
- About Me section
- Skills section
- Projects section
- Resume download
- Contact form
- Social media links

## Tech Stack

- HTML
- CSS
- JavaScript

## Live Demo

https://solankishweta.github.io/Personal-Portfolio/

## GitHub Repository

https://github.com/solankishweta/Personal-Portfolio.git

## Installation

1. Clone the repository
2. Open the project folder
3. Run locally by opening `index.html` in your browser

## Folder Structure

- `index.html`
- `style.css`
- `script.js`
- `images/`
- `portfolio/`
